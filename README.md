# Expansion-Cosmica-de-Hubble
Simulador de la expansión del Universo basado en la Ley de Hubble,  que establece que la velocidad de recesión de una galaxia es proporcional  a su distancia: v = H₀ · d (Hubble, 1929). El programa incluye un  catálogo dinámico de galaxias utilizando Programación Orientada a Objetos, con manejo de memoria dinámica, algoritmos de  ordenamiento, etc.
